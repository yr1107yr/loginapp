# loginapp

## ■ プロジェクト概要
本プロジェクトは、
**ユーザー名とパスワードを用いたシンプルなログイン認証機能**を実装したデモアプリケーションです。

ログイン時に入力された情報をデータベース内のユーザー情報と照合し、
認証が成功した場合は「ログイン成功」のメッセージを表示します。
**本デモでは、画面遷移は行いません。**

---

## ■ 主な機能
- ユーザー名・パスワードによるログイン認証
- 入力チェック（未入力／誤入力時のエラー表示）
- MySQL データベースを使用したユーザー情報管理
- パスワードのハッシュ化保存
- ログイン結果のフロントエンド表示（成功／失敗）

---

## ■ 使用技術
- フロントエンド：HTML / CSS / JavaScript
- バックエンド：Node.js
- データベース：MySQL
- その他：パスワードのハッシュ化による認証処理

---

## ■ 環境要件
- Node.js（v16 以上）
- MySQL（v8 以上）
- ブラウザ（Microsoft Edege）

---

## ■ 起動手順（ローカル環境）

1. MySQL を起動し、ユーザー管理用テーブルを作成
2. 付属のスクリプトを実行し、テスト用ユーザーを登録
3. Node.js サーバーを起動
4. ブラウザでログイン画面を開く
5. 事前登録済みユーザー情報を入力し、ログイン動作を確認

※ 本プロジェクトでは、新規ユーザー登録画面は実装していません。

---

## ■ データベース設計（概要）
- ユーザー情報は MySQL にて管理
- パスワードは **平文では保存せず、ハッシュ化した値**を保存

---

## ■ ロジック概要（ログイン処理）

1. フロントエンドでユーザー名・パスワードを入力
2. バックエンドへ入力値を送信
3. データベースから該当ユーザー情報を取得
4. 入力されたパスワードをハッシュ化
5. データベース内のハッシュ値と比較
6. 認証結果をフロントエンドへ返却
7. ログイン成功／失敗メッセージを表示
